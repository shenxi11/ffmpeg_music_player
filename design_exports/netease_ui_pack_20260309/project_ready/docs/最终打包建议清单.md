# 最终打包建议清单

## 推荐先放入项目的目录
- `icon/ui/base/`：基础和系统图标
- `icon/ui/player/`：播放器与歌词相关图标
- `icon/ui/list/`：列表操作、下载状态、异常提示图标
- `icon/ui/video/`：视频浮层与全屏控制图标
- `icon/ui/state/`：空状态、异常海报、欢迎背景
- `icon/ui/brand/`：Logo 浅色 / 深色 / SVG

## 页面级参考稿
- 首页、列表、播放页、视频页、设置页、插件页
- 个人中心、歌单详情、历史记录、下载管理、未登录态
- 搜索页、播放队列、歌词拖动预览、视频全屏浮层、登录错误态

## 组件级参考稿
- 基础控件、认证模块、推荐模块、系统反馈、主题对照

## 打包建议
- SVG 与 PNG@2x 一起保留，开发时优先引用 SVG。
- 先接入 `icon/ui` 与 `icon/ui/state`，页面设计稿可单独放入设计参考目录。
- 若项目中已有旧图标，可先按同名替换关键路径，再逐步替换页面视觉。
