# 项目落位方案

## project_ready 说明
- `auth/`：登录、欢迎、未登录、错误态相关页面稿与模块图
- `home/`：首页、搜索页、推荐模块参考图
- `library/`：本地列表、歌单详情、历史记录参考图
- `player/`：播放页、播放队列、歌词拖动与播放器控件参考图
- `video/`：视频页与视频图标状态图
- `settings/`：设置页、主题对照、控件状态矩阵与基础状态图
- `plugin/`：插件页、系统反馈、深色菜单/弹窗参考图
- `user/`：用户中心与下载管理参考图
- `state/`：关键空状态与异常海报
- `docs/`：接入映射、打包建议、qrc 候选清单
