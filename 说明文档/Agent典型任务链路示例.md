# Agent 典型任务链路示例（当前版）

## 1. 搜索并播放歌曲
1. 先搜索歌曲或歌手；
2. 由服务端完成对象解析；
3. 客户端接收结构化目标后执行播放；
4. `PlaybackViewModel` 更新当前播放与队列状态。

## 2. 搜索并加入歌单
1. 搜索歌曲；
2. 选择目标歌单；
3. 调用歌单添加能力；
4. 保留 canonical path，不走播放 URL 解析链。

## 3. 查询当前播放与队列
1. 读取当前播放快照；
2. 读取播放队列；
3. 返回当前标题、歌手、封面、进度和队列项。

## 4. 当前不推荐的链路
- 自动切换欢迎页模式；
- 自动修改用户名或上传头像；
- 在离线模式下强行发起在线动作。
