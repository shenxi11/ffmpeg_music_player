# Agent 工具清单与调用约定（当前版）

## 1. 文档目标
本文档用于说明当前 Agent 工具的调用约定，重点强调：工具应依赖当前代码已有业务入口，而不是凭文档推断出未实现能力。

## 2. 当前稳定工具主题
- 搜索歌曲 / 歌手 / 按歌手取歌
- 当前播放查询
- 播放控制
- 播放队列查询与管理
- 收藏与历史管理
- 歌单查询与管理
- 视频列表查询

## 3. 调用约定
1. 参数尽量使用结构化字段，而不是依赖自由文本在客户端做复杂解析；
2. 高副作用操作应在服务端或上层流程中先完成确认；
3. 遇到离线模式、未登录状态或页面能力不可用时，应允许客户端拒绝执行并返回边界说明。

## 4. 当前版本注意点
- 用户资料能力尚不建议直接作为开放工具默认暴露；
- 欢迎页模式切换不应视为普通业务工具；
- 加歌单操作需保留 canonical path。
