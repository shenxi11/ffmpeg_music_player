# UI视觉需求文档（网易云风格）

## 1. 项目背景与目标

### 1.1 软件定位
- 产品名称：云音乐（私域流媒体客户端，PC端）
- 使用场景：用户在家庭内网或私有网络连接自建服务器，在线播放音乐、视频、歌词、收藏、历史与推荐内容。
- 核心价值：本地设备轻存储、私域媒体统一管理、跨终端访问一致体验。

### 1.2 功能范围（本次UI改造覆盖）
- 欢迎与服务端验证页
- 登录/注册/忘记密码入口页
- 主界面导航（本地与下载、在线播放、推荐、最近播放、我喜欢）
- 音乐列表类页面（本地/在线/喜欢/历史）
- 播放控制栏（进度、播放模式、音量、桌面歌词、播放列表）
- 歌词页（沉浸态+相似推荐）
- 视频列表与视频播放窗口
- 设置页
- 插件页（音频转换、语音转写）

### 1.3 改造目标
- 统一视觉语言为“网易云风格”：简洁、轻拟物、红色强调、高信息密度但不拥挤。
- 保证高分屏清晰度（100%/125%/150% DPI）。
- 保证交互一致：悬停、按下、选中、禁用状态统一。
- 保证图片资源清晰、可扩展、可维护。

## 2. UI风格规范（网易云风格）

### 2.1 色彩系统
- 主色（品牌红）：`#EC4141`
- 主色悬停：`#F15050`
- 主色按下：`#C8342C`
- 主背景：`#F7F9FC`
- 卡片背景：`#FCFDFE`
- 深色播放背景：`#0F1115`
- 主文字：`#1E2430`
- 次文字：`#7A8499`
- 分割线/边框：`#DDE3ED`

### 2.2 形态语言
- 圆角优先：按钮 8/16/18，卡片 10/12，浮层 12。
- 轻边框+浅阴影，禁止厚重拟物。
- 重要操作仅用红色强调，避免多色竞争焦点。

### 2.3 字体与字号
- 中文字体：优先 `Microsoft YaHei UI`（Windows）
- 标题：18~20
- 正文：13~14
- 次信息：11~12
- 图标标签：12~14

### 2.4 动效规范
- 悬停过渡：120~180ms
- 弹窗出现：150~220ms（透明度+轻位移）
- 列表状态切换：不超过 200ms，避免“跳变”

## 4. 文件命名规范

### 4.1 图片资源命名（必须）
- 统一小写 + 下划线：
  - `module_element_state@scale.ext`
- 命名示例：
  - `player_btn_play_default@2x.png`
  - `player_btn_play_hover@2x.png`
  - `list_icon_favorite_active@2x.png`
  - `video_btn_fullscreen_default@2x.png`

### 4.2 目录规范
- `icon/ui/base/`：基础控件图标
- `icon/ui/player/`：播放控制相关
- `icon/ui/list/`：列表与操作图标
- `icon/ui/video/`：视频页图标
- `icon/ui/state/`：空状态插图
- `icon/ui/brand/`：品牌与Logo

### 4.3 状态后缀规范
- `_default` `_hover` `_pressed` `_active` `_disabled`

### 4.4 倍图后缀规范
- `@1x`（标准）
- `@2x`（主交付，必须）
- `@3x`（大屏优化，可选）

## 6. 图片质量与生成要求（重点）

### 6.1 清晰度要求
- 所有图标至少交付 `@2x`。
- 基础图标设计基准：24x24（导出 48x48）。
- 小图标不得低于 32x32 导出尺寸。
- 插图类资源最小边长不低于 1200px。
- 背景图建议 1920x1080 与 2560x1440 两档。

### 6.2 格式要求
- 图标优先：`SVG`（可缩放）+ `PNG@2x` 兜底。
- 插图/背景：`PNG`（无损）或高质量 `WebP`。
- 透明图层必须保留 alpha，禁止白底抠图。

### 6.3 风格一致性要求
- 线性图标线宽统一（建议 1.75~2.0px）。
- 圆角比例一致，避免同页出现不同风格。
- 光影强度统一，禁止部分图标过亮/过糊。
- 禁止拉伸变形，按原始比例导出。

### 6.4 可用性要求
- 深浅背景都可识别（需提供浅色版/深色版或自适应版本）。
- 在 125%/150% 缩放下不得模糊、锯齿、边缘发虚。
- Hover/Active 状态视觉差异必须明显。

### 6.5 验收标准
- 视觉验收：对照设计稿 1:1 检查。
- 技术验收：资源命名、尺寸、格式符合规范。
- 运行验收：主流程页面无错位、无图标糊、无状态缺失。
