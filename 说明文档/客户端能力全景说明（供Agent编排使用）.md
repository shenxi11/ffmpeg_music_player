# 客户端能力全景说明（当前版）

## 1. 能力总览
当前客户端已形成六类较稳定的能力面：
1. 启动与模式：欢迎页验证、离线直进；
2. 本地与下载：本地播放、下载列表、元数据补偿；
3. 在线音乐：搜索、推荐、在线播放、收藏、历史、歌单；
4. 播放与歌词：统一播放状态、队列、歌词显示；
5. 用户资料：登录、资料刷新、用户名修改、头像上传；
6. 视频：视频列表与视频播放。

## 2. 当前适合被 Agent/脚本利用的部分
- 搜索、播放、队列、收藏、历史、歌单、视频浏览等结构化能力。

## 3. 当前更适合人工主导的部分
- 欢迎页与服务器地址输入；
- 离线模式选择；
- 用户资料修改；
- 文件选择与本地资源上传。

## 4. 当前版本补充说明
与 4 月初之前相比，当前能力全景中新增了：
- 欢迎页离线直进；
- 个人主页与资料更新链；
- 下载页与本地列表的元数据补偿；
- 更统一的播放状态和封面动作。
