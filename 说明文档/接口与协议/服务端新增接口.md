# 服务端新增接口（当前客户端视角）

## 1. 文档目标
本文档用于汇总当前客户端版本已经依赖的“服务端新增接口”类别，避免继续沿用 3 月早期仅讨论在线状态或单一接口的旧文档。

## 2. 当前新增接口主题
### 2.1 欢迎页与客户端启动
- `/client/ping`
- `/client/bootstrap`

作用：服务验证、欢迎页进入主窗口的前置校验。

### 2.2 用户资料
- `GET /users/profile`
- 用户名修改接口
- 头像上传接口

作用：个人主页、顶部头像区和用户弹窗资料同步。

### 2.3 历史与收藏
- 批量删除播放历史
- 清空播放历史
- 收藏增删查接口

作用：历史页、用户主页最近播放预览、收藏页和主窗口动作联动。

### 2.4 私有歌单
- 获取歌单列表 / 详情
- 创建 / 更新 / 删除歌单
- 批量加歌 / 删歌
- 歌单排序

作用：歌单详情页、用户主页“创建的歌单”预览、列表页“添加到歌单”。

### 2.5 歌手检索
- `POST /artist/search`
- `POST /music/artist`

作用：歌手检索与按歌手获取音乐列表。

### 2.6 视频
- 视频列表接口
- 视频流地址接口

作用：视频列表页和独立视频播放窗口。

## 3. 当前客户端最关注的新增接口链
对当前客户端影响最大的新增接口有三组：
1. 资料链：profile / update username / upload avatar；
2. 歌单链：playlist CRUD + item 增删改排序；
3. 欢迎页链：client ping / bootstrap + 离线模式并存。

## 4. 当前版本结论
后续如果需要描述“服务端近期为客户端新增了哪些能力”，建议直接按本文档的六类主题归纳，而不要只引用单一接口说明。
