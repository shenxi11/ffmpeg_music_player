# 在线视频接口新增（当前版）

## 1. 文档目标
本文档用于说明当前客户端已经接入的在线视频相关接口能力，以及视频接口在当前系统中的实际使用方式。

## 2. 当前接口能力
### 2.1 视频列表
客户端通过视频列表接口获取：
- 视频标题 / 路径 / 封面 / 简要信息；
- 用于填充 `VideoListWidget` 与 `VideoListViewModel`。

### 2.2 视频流地址
客户端通过视频流地址接口获取：
- 可直接播放的视频 URL；
- 然后由 `VideoPlayerWindow` 调用 `loadVideo()` 进入播放。

## 3. 当前客户端调用链
1. `VideoListWidget` 进入页面后，请求视频列表；
2. 用户点击某个视频，客户端请求视频流 URL；
3. 拿到 URL 后打开 `VideoPlayerWindow`；
4. 窗口内部创建 `MediaSession` 完成播放。

## 4. 当前版本注意点
- 视频接口当前只服务“在线浏览与播放”，尚未与音乐主播放页完全融合；
- 客户端对视频接口的依赖集中在列表和流地址两类；
- 如果后续扩展视频推荐、视频历史或视频收藏，需要另行补文档和状态链说明。
