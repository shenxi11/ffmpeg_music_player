# Agent 联调提示词（Qt 客户端，当前版）

## 1. 使用目的
这份提示词用于在联调时提醒服务端 Agent：当前 Qt 客户端已经具备哪些稳定能力、哪些能力仍有边界，以及应如何理解当前版本的主链。

## 2. 当前联调时必须带上的上下文
- 客户端入口为欢迎页，可在线进入，也可离线直进；
- 主业务聚合中心是 `MainShellViewModel`；
- 播放状态真相源是 `PlaybackViewModel`；
- 用户资料页已经接通 profile、用户名修改和头像上传；
- 下载页和本地列表存在元数据补偿链；
- 离线模式下在线动作不可用。

## 3. 当前推荐提示词基线
- 先用结构化对象解析，再发客户端工具调用；
- 不要假设客户端脚本支持条件、循环、并发；
- 不要把欢迎页模式切换、用户资料编辑当成默认无保护动作；
- 查询播放态时优先依赖统一快照，而不是页面视觉状态。
