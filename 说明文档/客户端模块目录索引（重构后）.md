# 客户端模块目录索引（当前版）

## 1. 目录总览

### 1.1 C++ 目录
- `src/app`：应用启动与主窗口装配，包括欢迎页回流、离线模式、页面布局与全局连接。
- `src/audio`：音频解码、音频会话、共享播放器与缓冲控制。
- `src/video`：视频解码、视频缓冲、视频渲染与媒体会话。
- `src/playback`：音视频焦点与播放状态管理。
- `src/network`：HTTP 请求适配、网络服务层、响应缓存、在线状态。
- `src/download`：下载任务、下载管理、本地音乐缓存、下载列表模型。
- `src/lyrics`：歌词解析、桌面歌词与歌词编辑。
- `src/models`：`Music`、用户、视频等基础数据模型。
- `src/common`：配置、日志、封面查询、通用工具。
- `src/ui/widgets`：Widgets 层容器与页面宿主。
- `src/ui/qml_bridge`：QML 桥接对象。
- `src/viewmodels`：主壳、播放、登录、视频、在线列表、本地列表等 ViewModel。

### 1.2 QML 目录
- `qml/components/auth`：登录页、欢迎页相关视图资源。
- `qml/components/user`：顶部头像入口、用户弹窗、个人主页。
- `qml/components/search`：搜索框。
- `qml/components/library`：本地、下载、在线、推荐、收藏、歌单列表及其通用动作组件。
- `qml/components/playback`：播放页、队列、控制栏、进度条、播放历史。
- `qml/components/lyrics`：歌词显示与桌面歌词相关页面。
- `qml/components/video`：视频列表与视频播放页。
- `qml/components/settings`：设置页。
- `qml/theme`：主题与播放页样式 token。
- `qml/assets`：图标、默认封面、默认头像等资源。

## 2. 当前主窗口相关模块
### 2.1 `src/app`
- `main.cpp`：应用启动、欢迎页验证、离线直进与主窗口重入链路。
- `main_widget.cpp/.h`：主窗口壳层，负责页面装配、导航切换、模式控制。
- `main_widget.library_connections.cpp`：列表页、推荐、歌单、用户资料与主窗口连接。
- `main_widget.menu_auth.cpp`：登录、个人主页、欢迎页返回、头像资料相关连接。
- `main_widget.playback_connections.cpp`：播放页、列表封面动作、元数据回写连接。
- `main_widget.playback_list_handlers.cpp`：播放列表相关交互与离线模式保护。
- `main_widget.playback_state_handlers.cpp`：统一播放状态回写到各列表。

### 2.2 `src/viewmodels`
- `MainShellViewModel`：主壳业务聚合中心，负责账号、资料、推荐、收藏、历史、歌单、下载、本地缓存回写。
- `PlaybackViewModel`：播放状态唯一真相源，负责当前曲目、封面、歌手、位置、时长、队列快照。
- `LoginViewModel`：登录、注册、密码重置。
- `ServerWelcomeViewModel`：欢迎页服务端验证。
- `LocalMusicListViewModel` / `OnlineMusicListViewModel` / `VideoListViewModel`：对应页面的数据入口。

## 3. 当前页面宿主与桥接
### 3.1 用户与资料
- `src/ui/widgets/user/user_widget.*`：顶部头像与用户入口宿主。
- `src/ui/widgets/user/user_profile_widget.*`：个人主页 `QQuickWidget` 宿主。
- `src/ui/qml_bridge/user/userwidget_qml.h`：顶部头像入口与弹窗桥接。
- `qml/components/user/UserProfilePage.qml`：当前版本的用户主页。

### 3.2 列表与下载
- `src/ui/widgets/library/music_list_widget_local.*`：本地音乐列表宿主。
- `src/ui/widgets/library/music_list_widget_net.*`：在线音乐列表宿主。
- `src/ui/widgets/library/local_and_download_widget.*`：本地 / 下载 / 正在下载容器。
- `src/download/download_task_model.*`：下载列表直接消费的模型。
- `qml/components/library/DownloadedMusicList.qml`：下载音乐页。
- `qml/components/library/SongCoverAction.qml`：统一封面播放交互组件。
- `qml/components/library/SongActionStrip.qml`：统一列表动作条组件。

### 3.3 播放与歌词
- `src/ui/widgets/playback/play_widget.*`：播放详情页和控制链桥接。
- `src/ui/qml_bridge/playback/playlist_history_qml.*`：播放队列桥接。
- `qml/components/playback/ProcessSlider.qml`：当前主控制栏 QML。
- `qml/components/playback/PlayHistoryList.qml`：最近播放页。
- `qml/components/lyrics/LyricDisplay.qml`：歌词显示与播放页歌词卡片。

### 3.4 视频
- `src/ui/widgets/video/video_list_widget.*`：视频列表与视频播放入口。
- `src/video/*`：`MediaSession`、`VideoSession`、`VideoRendererGL` 等视频核心。

## 4. 当前高价值公共模块
- `src/common/settings_manager.*`：账号、资料、欢迎页相关持久化配置。
- `src/common/cover_lookup.*`：封面记忆查询。
- `src/download/local_music_cache.*`：本地歌曲封面/歌手/时长缓存。
- `src/network/httprequest_v2.*`：业务接口适配层。
- `src/network/online_presence_manager.*`：在线会话状态维护。

## 5. 当前主链路上的新增模块
与 3 月版相比，以下模块或职责是当前版本新增或显著增强的：
- `src/ui/widgets/user/user_profile_widget.*`
- `qml/components/user/UserProfilePage.qml`
- `src/common/cover_lookup.*`
- `MainWidget` 的离线模式收口方法与欢迎页返回链
- `DownloadTaskModel` 的 `artist` / `duration` role 与本地缓存补偿
- `PlayWidget` 本地元数据回写链（封面、时长、歌手）

## 6. 当前未作为主运行链的历史文件
下列文件仍可保留作历史对照，但不应再作为当前实现说明的主依据：
- `src/app/main_qml.cpp`
- 仓库中早期遗留的旧 QML/桥接原型文件
- 已不再承担主控制栏职责的旧 `ControlBar` / `CustomControlBar` 视觉路径

## 7. 结论
当前版本的客户端结构可以概括为：
- `MainWidget` 负责壳层与运行模式；
- `MainShellViewModel` 负责业务聚合；
- `PlaybackViewModel` 负责播放状态统一；
- QML 页面负责表现层和局部交互；
- 下载、本地缓存、用户资料与欢迎页流程均已收敛进当前主模块图谱。
