# 客户端模块目录索引（重构后）

## 1. 目录总览

### C++ 源码

- `src/app`：应用入口与主窗口装配。
- `src/audio`：音频解码、播放、会话与 PCM 管线。
- `src/video`：视频解码、渲染、会话与播放器窗口。
- `src/playback`：音视频焦点与播放状态管理（状态机相关）。
- `src/network`：HTTP 请求与网络服务层。
- `src/download`：下载管理、本地缓存与模型。
- `src/lyrics`：歌词解析、桌面歌词与歌词编辑能力。
- `src/models`：音乐/用户等基础数据模型。
- `src/common`：日志、配置、公共头文件。
- `src/ui/widgets`：Qt Widgets 层，按功能子目录拆分。
- `src/ui/qml_bridge`：QML 桥接层，按功能子目录拆分。
- `src/viewmodels`：ViewModel 层（当前主用 `PlaybackViewModel`）。
- `src/tools`：工具插件相关实现（如音频转换）。

### QML 资源

- `qml/components/auth`：登录界面。
- `qml/components/user`：用户头像/弹窗。
- `qml/components/search`：搜索框。
- `qml/components/library`：本地、下载、在线音乐列表。
- `qml/components/playback`：播放栏、进度条、播放历史、播放页。
- `qml/components/lyrics`：歌词与桌面歌词设置。
- `qml/components/video`：视频列表与视频播放页。
- `qml/components/settings`：设置页。
